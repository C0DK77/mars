# MARS — Multi-layer Aerial Recognition System

**Full-spectrum drone detection for military and critical infrastructure**

MARS is a multi-layer drone detection platform fusing six independent sensing modalities — EO/IR camera, acoustic, RF, radar, 5G/WiFi, and satellite — into a single fused track for military and critical-infrastructure operators.

## 🎯 Detection Layers

- **L1 — EO / IR Camera** — Day/night electro-optical and infrared detection with AI classification
- **L2 — Acoustic** — Microphone array with TDOA triangulation of rotor signatures
- **L3 — RF Spectrum** — Wideband SDR scanning of drone control links (2.4 GHz, 5.8 GHz)
- **L4 — UAS Radar** — Micro-Doppler radar for small RCS targets at up to 5 km
- **L5 — 5G / WiFi** — Passive bistatic detection using urban wireless infrastructure
- **L6 — Satellite R&D** — Bistatic radar exploiting LEO/Starlink downlink beams (research phase)

## 🔧 Technology

- Extended Kalman filter fusion engine
- On-edge neural classification (no cloud dependency)
- LINK-16 / ASTERIX / REST API C2 integration
- Modular, sensor-agnostic open architecture

## 🚀 Applications

- Military forward operating bases
- Nuclear and energy critical infrastructure
- Urban and event security
- Naval and maritime protection
